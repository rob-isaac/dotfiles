A Dotfile Repo based on https://www.atlassian.com/git/tutorials/dotfiles.

# Starting From Scratch:
```
git init --bare $HOME/.cfg
alias config='/usr/bin/git --git-dir=$HOME/.cfg/ --work-tree=$HOME'
config config --local status.showUntrackedFiles no
echo "alias config='/usr/bin/git --git-dir=$HOME/.cfg/ --work-tree=$HOME'" >> $HOME/.profile
```
then commit dotfiles with `config add <dotfile>; config commit -m <message>`

# Installing on new System
```
echo ".cfg" >> .gitignore
git clone --bare <git-repo-url> $HOME/.cfg
alias config='/usr/bin/git --git-dir=$HOME/.cfg/ --work-tree=$HOME'
config checkout # this may throw error for overwrite
config config --local status.showUntrackedFiles no
```
then commit dotfiles with `config add <dotfile>; config commit -m <message>`
